## Quick Setup

The fastest way to start a new project:

```bash
npx retend-start@latest
```

This creates a complete project with:

- **TypeScript** or JavaScript (your choice)
- **Vite** for fast development
- **Routing** for multi-page apps
- **Tailwind CSS** (optional)

### Adding Retend to an Existing Project

```bash
pnpm add retend retend-web
```

- **`retend`** — Core reactivity (Cells, Context, lifecycle)
- **`retend-web`** — DOM renderer for your pages

## Your First App

Here's a simple counter:

```tsx
import { renderToDOM } from 'retend-web';
import { Cell } from 'retend';

const App = () => {
  const count = Cell.source(0);

  const incrementCount = () => count.set(count.get() + 1);

  return (
    <>
      <h1>Hello, Retend</h1>
      <p>Click the button below to update the count.</p>

      <button type="button" onClick={incrementCount}>
        Counter: {count}
      </button>
    </>
  );
};

renderToDOM(document.getElementById('app')!, App);
```

**How it works:** Pass `{count}` directly into your JSX. Retend watches that Cell and updates the text automatically when it changes. The `App` function runs only once.

### Run Your App

```bash
pnpm run dev
```

Your app will be at `http://localhost:5229`.
