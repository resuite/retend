Welcome! This tutorial will help you start your first project with Retend, a modern JavaScript framework for building user interfaces. We focus on keeping your applications fast and your code simple.

## Prerequisites

Before we begin, please ensure you have the following installed:

- **Node.js**: A JavaScript runtime environment. If you don't have it, download it from the [official Node.js website](https://nodejs.org/).
- **npm** or **pnpm**: A package manager, which usually comes with Node.js.

We also assume you are familiar with basic HTML, CSS, and JavaScript.

## Setting Up Your First Project

The fastest way to start learning Retend is to scaffold a new project. Open your terminal and run this command:

```bash
npx retend-start@latest
```

This command automatically creates a complete project structure for you, including:

- [TypeScript](https://www.typescriptlang.org/) or JavaScript
- [Vite](https://vitejs.dev/) for an exceptionally fast development experience
- Built-in routing for multi-page applications
- [Tailwind CSS](https://tailwindcss.com/) (optional, for styling)

### Adding Retend to an Existing Project

If you prefer to add Retend to a project you already have, you can install the core packages directly:

```bash
pnpm add retend retend-web
```

- **`retend`**: Provides the core reactivity system (Cells, Context, and lifecycles).
- **`retend-web`**: Provides the DOM renderer to display your pages in a web browser.

## Writing Your First Application

Let us build a simple counter application. This will show you how Retend handles reactive data.

Create a new file and add the following code:

```tsx
import { renderToDOM } from 'retend-web';
import { Cell } from 'retend';

const App = () => {
  // 1. Create a reactive piece of data
  const count = Cell.source(0);

  // 2. Define a function to update the data
  const incrementCount = () => count.set(count.get() + 1);

  // 3. Return the user interface
  return (
    <>
      <h1>Hello, Retend</h1>
      <p>Click the button below to update the count.</p>

      <button type="button" onClick={incrementCount}>
        Counter: {count}
      </button>
    </>
  );
};

// 4. Mount the application to the browser's Document Object Model (DOM)
renderToDOM(document.getElementById('app')!, App);
```

### How the Counter Works

1. We create a `count` variable using `Cell.source(0)`. This wraps our number `0` in a reactive container.
2. We pass `{count}` directly into our user interface.
3. Retend watches that specific `Cell`. When you click the button and `incrementCount` runs, Retend automatically updates just the text inside the button. The `App` function itself only runs exactly once!

## Running Your Application

To see your application in action, run the development server:

```bash
pnpm run dev
```

Open your web browser and navigate to `http://localhost:5229`. You will see your counter ready to be clicked!
