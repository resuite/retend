# Getting Started

import { FeatureCard } from '@/components/FeatureCard';

Retend is a modern JavaScript framework for building user interfaces. It helps you create interactive web applications where the screen automatically updates whenever your data changes. Under the hood, Retend is designed to be incredibly fast and lightweight. It directly updates the parts of the page that change, avoiding unnecessary work and keeping your application quick and responsive.

## Installation

The easiest way to get started with Retend is to use the official project creation tool. This interactive tool will ask you a few questions and set up a complete, working project for you.

```bash
# Using npx (npm)
npx retend-start@latest
```

The `retend-start` tool will automatically configure:

- **TypeScript** or JavaScript (your choice)
- **Vite** for a blazing fast local development experience
- **Routing** for multiple pages, configured out-of-the-box
- **Tailwind CSS** (optional, but recommended for styling)

### Manual Installation

If you prefer to set up a project yourself, or if you want to add Retend to an existing project, you can install the core packages directly:

```bash
pnpm add retend retend-web
```

- **`retend`**: The core reactivity library containing Cells, Context, and Application Hooks.
- **`retend-web`**: The browser rendering engine that handles updating the DOM efficiently.

## Hello World

Let's take a look at a basic Retend application to see how these concepts come together. In Retend, your component functions only run **once**. Any data that needs to change over time is placed inside a `Cell`. When the `Cell`'s value is updated, the user interface updates automatically, without re-rendering the whole component.

```tsx
import { renderToDOM } from 'retend-web';
import { Cell } from 'retend';

const App = () => {
  // 1. Create a reactive value starting at 0
  const count = Cell.source(0);

  // 2. Create a simple function to update the count
  const incrementCount = () => count.set(count.get() + 1);

  // 3. Return what should be drawn to the screen
  return (
    <>
      <h1>Hello, Retend</h1>
      <p>Click the button below to update the count dynamically.</p>

      {/* 4. Bind our click handler and our reactive count Variable */}
      <button type="button" onClick={incrementCount}>
        Counter: {count}
      </button>
    </>
  );
};

// 5. Tell Retend where to put our application on the web page
const root = document.getElementById('app')!;
renderToDOM(root, App);
```

### Running Your App

If you used the `retend-start` command, you can start a local development server to see your application in your browser immediately. Simply navigate into your new project directory and run:

```bash
# Start the Vite development server
pnpm run dev
```

You should now see your Retend app running on `http://localhost:5229` (or the port shown in your terminal). Open that URL, click the counter button, and watch the number update instantly!
