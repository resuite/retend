# Changelog

### Core runtime and lifecycle

- Switched scope activation and pending setup flushes from timer delays to microtask scheduling, reducing lifecycle latency while preserving async activation order ([33cefe7](https://github.com/resuite/retend/commit/33cefe74fd2d3f96b73e897b944da984ea13c0f8)).
- Added regression coverage for `onSetup` lifecycle hooks so setup and activation timing behavior is guarded by tests ([784cc94](https://github.com/resuite/retend/commit/784cc9481c1b74e49fe634ef30a71c3b78922253)).

### Router and server rendering

- Fixed virtual DOM node replacement so moved replacement nodes are detached from existing parents, parent links stay consistent, and `replaceChildren()` does not leave duplicate ownership behind ([e23d3bd](https://github.com/resuite/retend/commit/e23d3bd1d3f499cb6d135865f177a43387017dcc)).
- Fixed router `replace()` to run navigation logic and ensured popstate navigation clears its navigation guard even when loading fails ([e23d3bd](https://github.com/resuite/retend/commit/e23d3bd1d3f499cb6d135865f177a43387017dcc)).
- Added regression tests for router title updates and virtual DOM node replacement behavior ([e23d3bd](https://github.com/resuite/retend/commit/e23d3bd1d3f499cb6d135865f177a43387017dcc)).

### Oxlint plugin and docs

- Added `retend/no-if-three-args` to reject legacy three-argument `If()` calls and true-only object bags, with guidance for direct true render functions and explicit true/false branch objects ([3d9ec3e](https://github.com/resuite/retend/commit/3d9ec3ef15c3de2dac3886af231a1907d40e215f)).
- Enabled `retend/no-if-three-args` in the recommended preset and migrated docs and examples away from true-only object bag syntax ([3d9ec3e](https://github.com/resuite/retend/commit/3d9ec3ef15c3de2dac3886af231a1907d40e215f)).

### Dependencies and maintenance

- Updated `retend` to depend on stable `@adbl/cells` releases, moving from the package preview URL to `^0.0.24` ([301e36e](https://github.com/resuite/retend/commit/301e36e6f129abaead5ab2b04f536e3aba088b0d), [dff35a2](https://github.com/resuite/retend/commit/dff35a23dc2f07570e58d15372cfe558a12ae7fb), [f9f193e](https://github.com/resuite/retend/commit/f9f193e03f976e5ba027b2d404a231663b1f3509)).
- Pruned stale lockfile entries, including unused experiment dependencies, after the dependency update ([616da52](https://github.com/resuite/retend/commit/616da521077a352131d0e8ce98330ea89e6753c0)).
