# Changelog

### Core runtime and rendering

- Fixed suspended scope activation and `Block` evaluation order so child nodes and inactive scopes are handled more predictably during rendering and lifecycle activation ([667e8dd](https://github.com/resuite/retend/commit/667e8ddb9a19520ec1d7a77a9a67872c98923637), [9a311c5](https://github.com/resuite/retend/commit/9a311c5c5d2002c6a98bddf5827b84ac551964c1)).
- Improved async derived-cell behavior by attaching listeners before reads, and added `useDerivedAsyncValue()` for deriving from static values, sync cells, and async cells ([f880155](https://github.com/resuite/retend/commit/f880155adc6c38db19e5b3e10f650290fd3ca325), [0cee4b7](https://github.com/resuite/retend/commit/0cee4b7d427662c81616bbbde119d5718c9a20ff)).
- Updated JSsX typing by adding SVG/CSS coverage for attributes such as `clipPath` and `opacity`, and removing `key` from the public JSX type surface ([949772a](https://github.com/resuite/retend/commit/949772ad7a33528d46c763775390e2c0e3fe2c8e), [6cb4e27](https://github.com/resuite/retend/commit/6cb4e270a6f30fa18941f4f294e56559ee5642a2)).
- Refined renderer movement and namespace handling by renaming `onBeforeNodeMove` to `onBeforeNodesMove`, improving SVG/MathML behavior, and adding correct HTML namespace handling for teleport targets ([25c880c](https://github.com/resuite/retend/commit/25c880c5748e66110f9725a6167c83f6d3d44892), [bc51766](https://github.com/resuite/retend/commit/bc517667c666f53dbf0af946ea9d65fdc238f358), [69529cb](https://github.com/resuite/retend/commit/69529cb5064b3f36fa007f7734294d4e14aec741)).

### Router

- Fixed several router navigation edge cases around failed matches, middleware redirects, redirect loops, and same-path redirects ([ac391aa](https://github.com/resuite/retend/commit/ac391aa234f8f6472f04438801dce935d7526a1a), [d3b057e](https://github.com/resuite/retend/commit/d3b057e17c7d151c7a78cb504850190e51e57b63), [1a00f32](https://github.com/resuite/retend/commit/1a00f32fa9a89f32ed7bfa66facb24af9172f990), [06a433f](https://github.com/resuite/retend/commit/06a433f56d11df7b23789ec322f0a66f66201a52)).
- Raised the default redirect limit to `100` to make legitimate nested redirects less likely to hit the guard rail too early ([6d47b72](https://github.com/resuite/retend/commit/6d47b7206ffc601f522fd4dac05819fe43cbbfc9)).
- Added locked-navigation handling that emits `routelockprevented`, restores the locked path after blocked browser history changes, and covers direct navigation, `replace()`, and window history with regression tests ([06a433f](https://github.com/resuite/retend/commit/06a433f56d11df7b23789ec322f0a66f66201a52)).

### Unique transitions

- Expanded `UniqueTransition` with `respectParentTransform`, `onStart`, `onEnd`, and `topLayer` options for better control over moved elements and animated ancestors ([2db8d53](https://github.com/resuite/retend/commit/2db8d53b21dc3ad1b01f58031d0577a634005b15), [db99e5f](https://github.com/resuite/retend/commit/db99e5f1d6caea0b0ad6d011efcc65482af5b27e), [0c2a04d](https://github.com/resuite/retend/commit/0c2a04d790104e90708719fdf419a0d23771f33f), [adfa456](https://github.com/resuite/retend/commit/adfa45614c8ba8aba9bb30d38eebda4befb4c4e1)).
- Reworked unique transitions from direct WAAPI setup to transient CSS stylesheets and per-element transition attributes, with cleanup for temporary styles, top-layer state, and transition markers ([ea5a8d7](https://github.com/resuite/retend/commit/ea5a8d7f34cd233141e9bdb758d72e5fb850320c), [26d869c](https://github.com/resuite/retend/commit/26d869c7256bca2f3ab0b6cc03ac812cb23814f7), [10fc74a](https://github.com/resuite/retend/commit/10fc74a5c8bb16ca843fac57145c5854b66e1cdd)).
- Simplified transition internals while improving duration parsing, teardown behavior, parent transform compensation, and animation restoration ([333220b](https://github.com/resuite/retend/commit/333220bf79647064d0e7278e4af2509b77e1d17a), [10fc74a](https://github.com/resuite/retend/commit/10fc74a5c8bb16ca843fac57145c5854b66e1cdd)).

### Oxlint plugin

- Added a reusable recommended preset export, allowing projects to extend `retend-oxlint-plugin/recommended.json` instead of manually enabling every Retend rule ([ffc54b2](https://github.com/resuite/retend/commit/ffc54b231cdf6e1262ac5ebcac828eae24eb2f37)).
- Added and documented more Retend-specific lint rules covering module cells, batched sets, task placement, teleport selectors, derived-cell writes, effect cleanup, raw ref callbacks, provider values, internal anchors, scope names, DOM ref access, and complex JSX handlers ([6c9b22c](https://github.com/resuite/retend/commit/6c9b22ce18b9d3445157a5626ed00d3f3008e591), [6eba2d1](https://github.com/resuite/retend/commit/6eba2d18090278344ca2314341cefa3d9e1c2d3c), [57d98b5](https://github.com/resuite/retend/commit/57d98b5d6ee6fadd6b5e06a11e6dfd32911bb5bc), [36833a6](https://github.com/resuite/retend/commit/36833a6055dfe70ae64690de54ac3b4929ff6be9)).
- Improved existing lint behavior for `Cell.task()`, props destructuring, and inline object type detection ([6e9c64c](https://github.com/resuite/retend/commit/6e9c64cc83c6734752f55a55c262cd078019a347), [6407e2b](https://github.com/resuite/retend/commit/6407e2b4927664d462499cdd1568ef89b11b9a79), [655393c](https://github.com/resuite/retend/commit/655393c82cfc3e5ea65466651d0e9be6c44d8b29)).

### Retend starter

- Refactored `retend-start` into dedicated modules for prompts, runtime checks, lifecycle, and project generation ([d411a4d](https://github.com/resuite/retend/commit/d411a4d873010775dd363ec553cd3b1bd54f35f8)).
- Generated projects now include the Retend oxlint preset, VS Code import organization and fix-all actions, Zed import sorting and fix-all actions, and editor extension recommendations ([ffc54b2](https://github.com/resuite/retend/commit/ffc54b231cdf6e1262ac5ebcac828eae24eb2f37)).

### Documentation and maintenance

- Updated docs for async utility hooks, rule validation architecture, lint rules, routing patterns, and generated project guidance ([0cee4b7](https://github.com/resuite/retend/commit/0cee4b7d427662c81616bbbde119d5718c9a20ff), [36833a6](https://github.com/resuite/retend/commit/36833a6055dfe70ae64690de54ac3b4929ff6be9), [ffc54b2](https://github.com/resuite/retend/commit/ffc54b231cdf6e1262ac5ebcac828eae24eb2f37)).
- Cleaned up project configuration and examples by ignoring `tmp`, removing unnecessary `key` props, and reducing production warning noise ([7d40927](https://github.com/resuite/retend/commit/7d409275fec11d12890b1279ae19d42ebeca9116), [de36678](https://github.com/resuite/retend/commit/de3667858936812323e1c31011e7e1ba42bf488d), [71111d9](https://github.com/resuite/retend/commit/71111d984ceb7960abb76b223bd3760194f201be)).
